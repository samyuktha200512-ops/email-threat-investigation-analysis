# Investigation Methodology

The investigation followed a structured SOC workflow to determine whether each email was legitimate or malicious.

## Methodology

1. Reviewed the email header information.
2. Verified sender, Reply-To, and Return-Path addresses.
3. Analyzed SPF, DKIM, and DMARC authentication results.
4. Examined the sending server and routing information.
5. Inspected email content for phishing and social engineering techniques.
6. Reviewed embedded URLs and file attachments.
7. Identified Indicators of Compromise (IOCs).
8. Performed OSINT verification using security tools where applicable.
9. Classified each email based on the collected evidence.
10. Documented findings, recommendations, and conclusions in a SOC-style report.
