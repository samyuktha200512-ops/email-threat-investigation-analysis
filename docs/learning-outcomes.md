# Learning Outcomes

Through this project, I gained practical experience in:

- Email header analysis
- Sender and domain verification
- SPF, DKIM, and DMARC validation
- Phishing and Business Email Compromise (BEC) detection
- IOC identification and documentation
- Email attachment and URL analysis
- Risk classification
- Professional SOC report writing
- Using OSINT tools to support investigations

This project strengthened my understanding of how SOC analysts investigate email-based threats using technical evidence and structured reporting.
